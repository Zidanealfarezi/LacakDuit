# LacakDuit
Aplikasi pencatatan keuangan pribadi berbasis web yang dibangun dengan React, Vite, dan Tailwind CSS. Memiliki fitur dashboard interaktif untuk melacak pemasukan dan pengeluaran harian serta kalkulasi saldo real-time. Proyek ini dibuat sebagai portofolio untuk mendemonstrasikan kemampuan Frontend Development modern.
